# STM32-K210-PanTilt-Tracker
STM32F103C8T6 intelligent pan‑tilt tracking system. Receives target coordinates from K210 host via YOLO detection, using serial protocol and state‑machine parsing. FreeRTOS multi‑tasking, PID‑controlled dual servos for real‑time tracking. OLED displays errors, with timeout detection and memory protection.
